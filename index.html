AI Fake Job Detection System

NLP + Bidirectional LSTM web application to classify job postings as real or fraudulent with real-time prediction and fraud probability output.

📂 GitHub
Python TensorFlow/Keras Bi-LSTM NLP NLTK Flask Glassmorphism UI
View full project details

Built an intelligent web application powered by Natural Language Processing and a Bidirectional LSTM neural network to classify job postings as Real or Fraudulent. Trained on 17,880 job records, the system analyzes linguistic patterns and gives instant predictions.

Designed to reduce job scam risk by automating authenticity checks that are impractical to do manually at scale.

  • Created full NLP pipeline: cleaning, tokenization, stopword removal, stemming, one-hot encoding, and sequence padding.
  • Built Bidirectional LSTM with Dropout regularization in TensorFlow/Keras.
  • Trained on 17,880 postings with 75/25 split for 12 epochs (Adam + binary cross-entropy).
  • Deployed using Flask with premium Glassmorphism UI and real-time result cards.
  • Output includes fraud probability score with clear red/green authenticity cues.
